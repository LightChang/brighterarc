## 承諾原文

要求各機關落實維護工區施工安全，並納為工程施工查核重點，相關強化工地職業安全管理作為如下：依「政府採購法」落實職業安全源頭管理。

## 來源回覆

**主旨**：（三十三）行政院函送李委員昆澤就第10屆第4會期施政質詢之書面答復，請查照案。

李委員就有關高雄捷運黃線行政程序等問題所提質詢，經交據有關機關查復如下：一、關於高雄捷運黃線行政程序問題：行政院業於民國108年5月24日核定高雄市政府所報「高雄捷運都會線（黃線）可行性研究報告」，高雄市政府續於109年3月31日將綜合規劃報告書報請交通部審議。交通部經書面審查、現勘、初審會議及本（110）年7月14日召開捷運審查委員會議審查通過，並於本年9月29日函報行政院。案經行政院於本年10月4日將「高雄捷運都會線（黃線）綜合規劃報告書」交議國發會審議，該會已於本年12月20日召開委員會議，俟該會函復審議意見後，行政院將儘速核處。二、關於觀光發展基金財源問題：(一)交通部觀光局（以下簡稱觀光局）觀光發展基金主要收入來源為「機場服務費」，受疫情影響，實施國境管制，造成「機場服務費」嚴重短收，形成收支短絀。惟其因應嚴重特殊傳染性肺炎（COVID-19）疫情辦理之紓困及振興措施主要係由特別預算撥入該基金協助觀光產業。另該基金截至本年10月底止，現金餘額尚有11億元，暫無資金短缺之虞。(二)為確保該基金主要支出「Tourism 2025臺灣觀光邁向2025方案」（以下簡稱T2025方案）能順利推動，經行政院本年4月6日同意朝「舉借」方式籌措財源。觀光局於本年4月20日函報「觀光發展基金自償性債務舉借及償還計畫案」至財政部審議，經該部於本年6月23日函復「原則同意」（本年至114年舉借上限151.24億元）。經評估本年下半年及111年上半年基金缺口59億元，於本年8月11日辦理基金短借公開比價作業，並於本年9月13日完成簽約。(三)未來推動T2025方案，將以撙節支出、量入為出原則，滾動檢討各項計畫之輕重緩急，以移緩濟急、減列工項及調整預算辦理，如有預算缺口，除滾動檢討以舉借方式籌措財源外，行政院亦將視該基金整體執行及資金需求情形，適時予以協助。三、關於春節防疫旅館準備問題：(一)為因應111年農曆春節入境旅客入住防疫旅館需求，觀光局相關作為如下：1.邀集6都、陸委會、衛福部及交通部民航局相關機關與會討論，瞭解春節前防疫旅宿供需情形；督促各地方政府積極輔導轄內旅宿業者增加量能，並掌握各地方量能增加狀況、已滿房地方之因應方式、不合理漲價之處置作為等。2.發布新聞稿鼓勵旅館業者及透過公會邀集旅館業者加入防疫旅館，為防疫共同貢獻心力。3.本年11月5日修正發布「交通部觀光局獎助直轄市及縣（市）政府推動溫馨防疫旅宿實施要點」，延長補助期限至111年6月30日，增加旅宿業者加入防疫旅館誘因。4.推動合意短期防疫旅宿措施，並於本年11月25日公布相關補助計畫，俾利地方政府儘速進行媒合。(二)觀光局已於本年11月11日設置「防疫旅宿查詢平臺」，公布本年12月14日至111年1月31日地方防疫旅宿可訂房情形資訊，彙整揭露各地方政府調查所轄防疫旅宿空房情形，以紅黃綠燈號顯示概念便利民眾查詢較有空房之週期，且每日更新，以做為選擇防疫旅館之參考；另已製作春節防疫旅宿訂房注意事項向民眾宣導，提醒民眾依「確認、查詢、訂房、填寫」相關步驟，事先做好防疫旅宿訂房準備，以順利入境完成居家檢疫程序。四、關於改善網路詐騙狀況問題：(一)行政院國家資通安全會報下設「網際犯罪偵防體系」，由相關機關成立「防治網路犯罪組」，包含內政部、法務部、經濟部、科技部、金管會、通傳會等機關，共同研析防制網路犯罪因應對策。(二)因網際網路內容包羅萬象，政府為促進網際網路內容安全，已於行政院國家資通安全會報-資通訊環境及網際內容安全組107年11月13日第2次會議，律定「網際網路內容管理基本規範及分工原則」，明確規範我國網際網路內容之管理與實體社會之管理方式相同，由各法令規定之主管機關依權責處理。(三)為與其他部會協力防範網路詐騙，通傳會除為行政院治安會報、內政部網際犯罪偵防體系協商平臺、法務部打擊跨境詐騙跨部會平臺、臺灣高等法院檢察署打擊坊間非法竊聽督導小組等跨部會平臺成員外，並與內政部警政署刑事警察局（以下簡稱刑事局）成立「防制電信詐欺與網路犯罪工作小組」，不定期邀集電信業者召開會議，研析電信詐騙之技術手法，並督導電信業者配合採取相關防制對策。目前已建立之防制電信詐騙重要措施如下：1.修訂電信業務系統網路技術規範，加強電信設備防制詐騙電話功能。2.國際話務於國內傳遞，受話端市話須顯示00X（我國國際冠碼002、005、006、007、009等），手機須顯示「+」號，方便民眾區別境外電話。3.督導電信業者建置NGN網路，並配合刑事局將境外詐騙電話進行攔阻。107年至109年分別攔阻1966萬通、1086萬通、916萬通，本年至9月底為761萬通。4.督導電信業者配合刑事局發送反詐騙宣導簡訊，提升民眾防範詐騙認知。107年至109年分別發送5962萬則、6201萬則、6398萬則，本年至9月底為4403萬則。5.督導電信業者統一國內簡訊來電顯號格式，去除附加之「+886」、「886」、「+」等，以利民眾分辨國內簡訊與境外電話之來電差異。6.強化外籍人士申請預付卡之行政管制，督導電信業者就外籍來臺人士於境內申辦預付卡時，須確實核對雙證件及其居留期間事實，販賣適合居留期間（短效期）之預付卡類型。(四)按財團法人台灣網路資訊中心（TWNIC）目前已建立一套網域名稱系統回應政策區域（Domain Name System Response Policy Zone DNS RPZ）機制，用於限制境內外惡意網域名稱或IP位址被用戶接取。因限制接取網域名稱或IP位址，涉及限制人民通信自由之重大爭議，啟動程序須建立在合法且正當之基礎。是以，TWNIC現階段受理RPZ之作業依據，為法院判決、裁定或行政機關依法（如「兒童及少年福利與權益保障法」第46條，以及「動物傳染病防治條例」第38條之3）裁處之命令。(五)為防止兒少接觸有害其身心健康之網路內容，通傳會依「兒童及少年福利與權益保障法」第46條規定，邀集衛福部、內政部、教育部、經濟部及文化部等目的事業主管機關，共同委託民間團體成立網路內容防護機構（iWIN），辦理法定7大任務，其中包含受理民眾申訴有害兒少身心健康之不當網路內容。iWIN如受理涉及網路詐欺之申訴案件，將引導民眾向內政部警政署設置之165反詐騙諮詢專線尋求協助。(六)通傳會研議之「數位通訊傳播服務法」草案，規劃介接各部會之作用法，以跨部會協力處理網路違法內容為原則，並提升網路平臺透明性與可問責性，一頁式詐欺事件偵辦機關如認該內容違法，並據以做成行政處分，網路平臺則當循相關義務程序配合移除下架，藉以維護網路環境之安全。該草案刻正於內部研議與凝聚共識中，待該會委員會議通過後，將依法預告法案內容。五、關於改善公共工程工安問題：(一)要求各機關落實維護工區施工安全，並納為工程施工查核重點：行政院工程會業於本年10月7日函請各機關除應加強控管工程品質及進度外，更應落實安全第一，以維護工區施工安全，相關強化工地職業安全管理作為如下：1.依「政府採購法」落實職業安全源頭管理：108年5月22日修正公布之「政府採購法」，增訂第70條之1，要求機關於工程規劃、設計，應依工程規模及特性，分析潛在施工危險，編製安全衛生圖說及規範，並量化編列安全衛生費用，且將設計成果納入招標文件。前項設計成果納入招標文件，並於招標文件規定廠商須依職業安全衛生法規，採取必要之預防設備或措施，實施安全衛生管理及訓練，使勞工免於發生職業災害，以確保施工安全。2.辦理施工風險評估：勞動部本年2月17日修正「營造工程施工風險評估技術指引」，工程主辦機關、設計及施工廠商可依該指引內容於公共工程全生命週期，如工程設計、施工規劃等階段實施施工風險評估。3.施工前職安檢查：依據行政院工程會工程採購契約範本附錄2工地管理，廠商每日施工前應辦理勤前教育與危害告知、勞工保險資料與教育訓練及個人防護具之檢查，並記載於施工日誌及回報監造單位，於監造報表確認檢查結果。4.加強檢驗停留點查驗：監造單位及施工廠商確實辦理檢驗停留點（含安全衛生事項）之各項施工抽查及材料抽驗工作，確認合格後才繼續下一階段施工，並透過施工查核及主辦機關督導，落實執行，確保工程品質及安全。5.緊急應變作為：工程團隊應隨時留意交通部氣象局氣象即時資訊，以及行政院農委會水保局之土石流防災資訊，並於豪雨、風災來襲前後及地震發生後，針對公共工程之結構、周遭環境、施工機具設備與相關安全設施應予檢查，如有安全顧慮，應立即採取相關對策，避免工地發生危害。6.強化主管機關監督並建立橫向聯繫機制：工程主管機關之施工查核小組除將工地安全衛生納為查核重點，如有嚴重缺失，需加重扣點，並通知當地勞動檢查機構，列為勞動檢查重點，與跨部會建立橫向聯繫機制，強化工地安全監督。(二)工程施工查核常見缺失之檢討精進作為：1.定期函送常見缺失態樣，責成各主管機關加強管理：自本年起，行政院工程會每季檢送「全國工程施工查核小組常見缺失態樣統計表」，請各主管機關督導所屬工程單位，依所屬工程類別及特性，研擬常見缺失之預防改善對策，避免缺失重複發生，確保工程施工品質。2.加重施工查核常見缺失之扣點計罰：本年7月修正「工程施工查核小組查核品質缺失扣點紀錄表」，針對施工查核常見及重複發生之安全衛生與工地管理缺失加重記點，促使廠商對於工地工安與施工管理之重視。(三)行政院工程會未來仍將持續要求各機關透過工程施工查核，強化並監督各工程主辦機關落實工程品質、進度及安全之源頭管理，確保公共工程於安全無虞前提下如質如期完成。
